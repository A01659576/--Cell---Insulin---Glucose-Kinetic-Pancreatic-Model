# --Cell---Insulin---Glucose-Kinetic-Pancreatic-Model
This is the repository with the codes created for the model β-Cell - Insulin - Glucose Kinetic Pancreatic Model.



## Project Files

- **Bitácora**: Contains all the feedback received.  
- **Artículo**: Clean version of the final article.  
- **Reto Optimización**: File used for research and to document all procedures, which are later included in the article.
- **HandMadeProcedure**: File with the hand made calculus.
- **Reto (Jupyter Notebook)**: Contains the code based on hand calculations; used to verify that the Phase 2 results are correct.
